BUILD NOTES

The examples in this directory are written using GLG Standard API and
can be linked with the GLG Standard API Library.

To compile the C++ version of example, define the GLG_CPP_STANDARD_API 
preprocessor definition.

On Linux/Unix:

    Use compilation option: -D GLG_CPP_STANDARD_API

On Windows: 

    Add GLG_CPP_STANDARD_API to the Preprocessor Definitions in the 
    Visual Studio project.
